HEAVAN WEBSITE – PROJEKTORDNER

Inhalt:
- index.html = Startseite
- ueber.html = Über Heavan
- modelle.html = Unsere Modelle
- ausbau.html = Individueller Ausbau
- styles.css = gesamtes Styling
- script.js = mobiles Menü
- assets/ = alle verwendeten Bilder und das Logo
- references/ = die gelieferten PNG-Mockups als Designreferenz

SO STARTEST DU DIE WEBSITE:
1. ZIP entpacken
2. Ordner in Visual Studio Code öffnen
3. index.html öffnen
4. Mit der Erweiterung "Live Server" starten

WICHTIG:
- Die Schrift Inter Tight wird per Google Fonts geladen.
- Falls du offline arbeitest, wird automatisch eine Ersatzschrift verwendet.
- Alle Bildpfade sind bereits in den HTML-Dateien eingebunden.

TIPPS FÜR ÄNDERUNGEN:
- Texte ändern: direkt in den HTML-Dateien
- Farben, Abstände, Größen ändern: in styles.css
- Neue Bilder austauschen: im Ordner assets/ ersetzen und Dateinamen gleich lassen
